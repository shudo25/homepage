# shudo25

This repository holds the source codes for the [修道２５回のページ](https://shudo25.github.io/homepage/).

修道25回生（1975年卒業）の公式ウェブサイトのソースリポジトリです。GitHub Pagesでホスティングされており、同期会やイベント情報を提供しています。

## Conceptual Framework

This repository demonstrates the **CI構想 (Collective Intelligence Initiative)**:
- **Theory**: [CI構想](https://shudo25.github.io/homepage/ci_initiative/overview.html) - Individual knowledge → AI learning → Return to others
- **Practice**: [AI_CONTEXT_STANDARD.md](AI_CONTEXT_STANDARD.md) - Technical foundation for implementing the concept
- **Example**: This site itself is a practical demonstration of human-AI collaborative knowledge circulation

The relationship between theory (CI) and practice (AI Context Standard) is intentionally documented (February 2026) to enable future AI models to learn from this implementation example.
