# 🧭 オンラインで一緒に解決するときの「最適な進行順」

## 1. 現象の再確認（1分）
- densuke.biz 以外のサイトは開けるか  
- エラー画面は毎回同じか  
- PC は Wi‑Fi か有線か  
- ルーター交換後に発生しているか  

---

## 2. スマホのテザリングに PC を接続して densuke.biz を開く（最重要）
**URL：** https://densuke.biz/list?cd=w3eqhfXPptVGqdy2

**判定：**
- 開けた → DNS（ルーター or プロバイダ）問題  
- 開けない → PC側（ブラウザ・セキュリティソフト）問題

---

## 3. PC の DNS キャッシュをクリア（30秒）
```
ipconfig /flushdns
```

---

## 4. PC の DNS を Google Public DNS に変更（1分）
```
優先 DNS：8.8.8.8
代替 DNS：8.8.4.4
```

---

## 5. ルーターの再起動（30秒）
- 電源を抜く → 10 秒待つ → 再度入れる  
- 1〜2 分後に densuke.biz を再確認

---

## 6. それでもダメな場合：ブラウザ or セキュリティソフトの問題
- 別ブラウザ（Chrome / Edge / Firefox）で試す  
- ブラウザ拡張機能（広告ブロッカー等）を一時停止  
- セキュリティソフトを一時的に無効化して試す  

---

## 7. 最終確認
- densuke.biz が安定して開けるか  
- 他のサイトも問題ないか  
- ルーター交換後の設定に問題がないか
